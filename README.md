## Usage
Clone and install dependencies.
```
git clone https://github.com/pearllaw/react-flash-cards.git
cd react-flash-cards/
npm install
```
Create a '.env' file in the project root. Example:
```
PORT=3000
```
Run the app
```
npm run watch
```
